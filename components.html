04 — Product

Components

Every component below is lifted directly from the live FaithStream product code — not redesigned, just documented and annotated against the token system, so new screens reuse these patterns instead of building slightly-different versions each time.

All components below are live product code, unchanged

Buttons

Three button styles, each with a distinct job. None compete with each other for attention because each one only appears in its own context.

Badges

Content metadata — rating, genre, duration — shown as small pills. The rating badge is the only one that uses the magenta accent; everything else stays neutral so it doesn't compete for attention.
